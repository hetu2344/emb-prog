- What is embedded system? A computer system that performs a perticular task inside a machine or a large electrical system
- It performs perticular/specific tasks.
- The embedded system is generally stored in the memory of the system whose behavious is fixed.
- Components of Embedded system: Microcontrollers (adruino uno), sensors (temperature) and actuators (motor), software
- Examples of embedded system: Microwave, airpods, cars, washing machine
- Requirements of embedded system: Need to response in certain time, runs 24/7
- Constrains of embedded system: Limited memory and space, runs on battery
