## Recap of C
- data types: char, int, double, float, short
- <stdint.h> data types: int8_t, uint8_t, int16_t...
- usually char is 8 bits and int is 32 bits in size.
- struct: allows us to encapsulate multiple data together
- enum: enumerated data types
- union: overlapped storage over multiple data types
- Use od struct, enum and union is not compulsory, but by using them we can make the code more readeable and maintainable
